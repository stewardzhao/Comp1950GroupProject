CSS3 promises to provide much greater control over the 'experience layer' of a user's interaction with a web page, and also to address some of the problematic layout issues of CSS2. Technically, the current W3C recommendation for web developers and browser vendors is CSS2. However, if close attention is payed to browser support, many CSS3 features can be effectively deployed today.
